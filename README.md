# 待辦事項管理排程系統

## Proposal Report

### 動機與目標
在日常生活中有許多的待辦事項，我們需要記住並根據每項的重要程度、期限來安排先後。因此做出這個工具紀錄各事項，並根據事項的特性來安排順序，減少這些瑣事帶來的勞心勞力。

### 競品比較
市面上的類似競品如Microsoft To Do，已具備相當完整的功能。然而這些產品主要仍聚焦於「任務管理效率」，較少針對「使用者動機維持」進行設計。
本專案新增的「獎勵機制」則補足了這一點差異。使用者可為每個待辦清單設定完成後的獎勵內容(如休息時間、娛樂活動或個人獎勵)，當該清單內所有任務完成時，系統會自動跳出獎勵提示，提供即時且具情緒回饋的正向強化。
使用「獎勵機制」能夠讓使用者聚焦在任務上，提高完成任務的意願，正向回饋也有助於提高繼續使用本產品的意願。除了自我激勵，也能適用於家長以獎勵激勵孩子完成任務(如家事、功課)等情境。

### 預期功能
- 新增/刪除待辦事項（名稱、截止時間、優先級） 
- 顯示所有待辦事項  
- 標記事項為已完成  
- 根據優先級與截止時間自動排序  
- 顯示目前最需要優先完成的任務  

### 使用技術
- 程式語言：Python  
- 開發工具：VS Code  
- 資料儲存：JSON 或 TXT  
- 介面：CLI
